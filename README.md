# create-x509-cert-example
opensslでX.509のSSLサーバー証明書を生成するサンプルアプリケーションです。
ルート証明書と中間証明書と末端証明書の生成をできるようにする予定です。

証明書の有効性をdocker-compose + ブラウザでチェックすることができます。
例えば、SSLの仕様が変わったときにチェックすること、証明書の有効性を確認することができます。

## firefoxに証明書を登録
### ブラウザにルート証明書を登録
設定 → プライバシー → セキュリティ → 証明書を表示 → 認証局証明書タブ → 読み込む → exportKey/root.x509.pem
→ 「webサイトを信頼する」のみにチェックを入れて登録

### ブラウザにクライアント証明書を登録
設定 → プライバシー → セキュリティ → 証明書を表示 → あなたの証明書タブ → 読み込む → exportKey/leafForClient.p12
→ passwordを入力（generate.shに-passoutを入れている文字列）
→ ブラウザでリロード（https://localhost） → このサイトはあなたの証明書を求めています → OKを押す
→ レスポンスが返ってくる
